What I've Built

Featured Project
01

SmartRoute — AI-Powered Traffic Congestion Predictor

An end-to-end machine learning system that predicts urban traffic congestion from real-world transit parameters. Built, trained, and deployed solo — from raw data to live interactive map.

Python scikit-learn Random Forest FastAPI React.js Tailwind CSS Mapbox 3D Supabase PostgreSQL JWT Joblib Seaborn
  • Built and compared 4 ML models (Linear Regression, Decision Tree, Random Forest, Gradient Boosting) — Random Forest won with best accuracy, lowest MAE and RMSE
  • Engineered full data preprocessing pipeline: missing value handling, StandardScaler normalization, categorical encoding, train-test split
  • Deployed trained model as FastAPI REST backend with React + Tailwind CSS frontend featuring live Mapbox 3D congestion overlays
  • Implemented JWT authentication via Supabase (PostgreSQL), modular model storage, and data visualization with Matplotlib & Seaborn
  • Inputs: vehicle volume, time of day, weather, road capacity — outputs: color-coded congestion level on interactive 3D map
